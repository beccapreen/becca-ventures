# 🌙 Cosmic Daily — App Concept

**Status:** Concept stage · March 2026

A beautifully designed iPhone lock screen widget and app that gives you a daily spiritual weather report — moon phases, cosmic events (Mercury retrograde, Neptune seasons, eclipses), daily affirmations, and grounded guidance.

[View Concept Dashboard →](https://beccapreen.github.io/becca-ventures/cosmic-daily/)

## Core Features
- Lock screen widget (moon phase + daily affirmation)
- Cosmic weather in plain English
- Daily guidance card
- Lunar calendar with new/full moon prompts
- Optional birth chart personalisation
